# ClipNote 密码填充扩展

这是 ClipNote 随桌面端源码交付的 Chrome/Edge Manifest V3 扩展。

## 安装

1. 在 Chrome 打开 `chrome://extensions`，或在 Edge 打开 `edge://extensions`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择本目录。
4. 解锁 ClipNote 密码本，在“密码本设置 → 浏览器自动填充”中复制配对码。
5. 打开扩展，粘贴并保存配对码。

## 使用

在 HTTP/HTTPS 登录页面打开扩展。扩展会优先显示与当前域名匹配的条目，也可输入名称、账号、网址或标签进行模糊搜索。点击一个候选后，扩展会填入页面中可见的账号和密码输入框，不会自动提交表单。

本机服务监听 `127.0.0.1:32145`。密码本锁定时扩展不会返回候选或凭据；在桌面端换新配对码后，需要在扩展中保存新的配对码。
