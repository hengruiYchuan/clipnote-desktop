# ClipNote V0.1 Editorial Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a runnable Windows Tauri desktop shell that demonstrates ClipNote’s approved Editorial Desk identity through a docked edge tab, an animated workspace, realistic clipboard/note previews, tray controls, a global shortcut, accessibility, and automated verification.

**Architecture:** React owns the visual state machine (`collapsed`/`expanded`, active section, search, motion preference), while Rust owns window geometry, tray, and global-shortcut actions. A narrow typed `desktopBridge` is the only frontend/native boundary, and browser tests use an in-memory bridge so the complete shell can be exercised without launching Tauri.

**Tech Stack:** Tauri 2.11, Rust 1.97, React 19, TypeScript 7, Vite 8, CSS design tokens, Motion 12, Zustand 5, Lucide React, Vitest 4, Testing Library 16, Playwright 1.61, pnpm 11.

---

## Scope boundary

This plan is the first independently testable slice of the approved product. It deliberately ships a polished shell with fixture content before adding live clipboard persistence. It does **not** implement SQLite, clipboard listeners, FTS5, note persistence, privacy filters, AI calls, installers, or auto-update; those receive separate plans after this shell is approved in the running desktop app.

```ts
import { create } from "zustand";

export type ShellMode = "collapsed" | "expanded";
export type ShellSection = "recent" | "favorites" | "notes";

type ShellState = {
  mode: ShellMode;
  section: ShellSection;
  query: string;
  expand: () => void;
  collapse: () => void;
  toggle: () => void;
  setSection: (section: ShellSection) => void;
  setQuery: (query: string) => void;
};

export const useShellStore = create<ShellState>((set) => ({
  mode: "collapsed",
  section: "recent",
  query: "",
  expand: () => set({ mode: "expanded" }),
  collapse: () => set({ mode: "collapsed" }),
  toggle: () => set((state) => ({
    mode: state.mode === "collapsed" ? "expanded" : "collapsed",
  })),
  setSection: (section) => set({ section }),
  setQuery: (query) => set({ query }),
}));
```

- [ ] **Step 5: Implement the typed bridge**

Create `src/bridge/desktopBridge.ts`:

```ts
import { invoke } from "@tauri-apps/api/core";

declare global {
  interface Window { __TAURI_INTERNALS__?: unknown }
}

const isTauri = () => typeof window !== "undefined" && "__TAURI_INTERNALS__" in window;

async function call(command: string) {
  if (isTauri()) await invoke(command);
}

export const desktopBridge = {
  expand: () => call("expand_main_window"),
  collapse: () => call("collapse_main_window"),
  toggle: () => call("toggle_main_window"),
};
```

```rust
use tauri::{
    menu::{Menu, MenuItem},
    tray::{MouseButton, MouseButtonState, TrayIconBuilder, TrayIconEvent},
    App, Manager,

pub fn install(app: &mut App) -> tauri::Result<()> {
    let open = MenuItem::with_id(app, "open", "打开 ClipNote", true, None::<&str>)?;
    let pause = MenuItem::with_id(app, "pause", "暂停采集", true, None::<&str>)?;
    let quit = MenuItem::with_id(app, "quit", "退出", true, None::<&str>)?;
    let menu = Menu::with_items(app, &[&open, &pause, &quit])?;

    let mut tray = TrayIconBuilder::new()
        .menu(&menu)
        .show_menu_on_left_click(false)
        .on_menu_event(|app, event| match event.id.as_ref() {
            "open" => { let _ = crate::window::expand_main_window(app.clone()); }
            "pause" => { let _ = app.emit("capture-state-changed", false); }
            "quit" => app.exit(0),
            _ => {}
        })
        .on_tray_icon_event(|tray, event| {
            if let TrayIconEvent::Click { button: MouseButton::Left, button_state: MouseButtonState::Up, .. } = event {
                let _ = crate::window::expand_main_window(tray.app_handle().clone());
            }
        });
    if let Some(icon) = app.default_window_icon() { tray = tray.icon(icon.clone()); }
    tray.build(app)?;
    Ok(())
}
```

```rust
use tauri::App;
use tauri_plugin_global_shortcut::{Code, GlobalShortcutExt, Modifiers, Shortcut, ShortcutState};

pub fn install(app: &mut App) -> tauri::Result<()> {
    let toggle = Shortcut::new(Some(Modifiers::ALT), Code::Space);
    app.handle().plugin(

            .build(),
```
